### Общие положения по валидации полей

| Поле | Формат | Обязательно | Примечания                                                                                                                                                        |
|------|-------|------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Номер карты | 16 цифр (только цифры) | ✅ | Пробелы разрешены                                                                                                                                                 |
| Месяц | 01–12 (2 цифры) | ✅ | —                                                                                                                                                                 |
| Год | 2 цифры (текущий год – текущий год + 5) | ✅ | При значении вне диапазона:&#10; — меньше текущего года → «Истёк срок действия карты»;&#10; — больше текущего года + 5 лет → «Неверно указан срок действия карты» |
| Владелец | Только латиница и пробелы | ✅ | Не допускаются:&#10; — цифры; &#10; — кириллица; &#10; — спецсимволы                                                                                              |
| CVC | 3 цифры | ✅ | Не принимает:&#10; — значения более 3 цифр; &#10; — буквы; &#10; — спецсимволы; &#10; — отрицательные значения; &#10; — пустое значение                                                                                                                      |
